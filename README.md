# 💻 프로젝트 소개

#### **🦃 터키 익스프레스(Turkey Express) 🏍️**

#### - 한식, 중식, 양식 등 다양한 음식을 주문받고 배달할 수 있는 서비스입니다.

#### - 경쟁력 있는 높은 할인율과 저렴한 수수료를 통해 고객과 가맹점 모두 만족할 수 있는 플랫폼을 제공합니다.

#### - 사용자 친화적인 인터페이스와 신속한 배달 시스템으로 최상의 경험을 제공합니다.
